# SteamAnalytics

## Objective
    1. Detected fake reviews
    2. Conducted sentiment analysis and built topics model to analyze user reviews in some popular game genres
    3. Developed game recommendation method based on product similarity and user similarity

## Data Description
1. Game dataset
    - AppID
    - AppName
    - Genre
    - Tags
    - Developer
    - Publisher
    - ReviewInfor
    - ReviewStatus
    - ReleaseDate
    - IsFree
    - Price
      
2. Game reviews dataset
    - SteamID 
    - NumGamesOwned
    - NumReviewsGiven
    - PlayTime
    - PlayTimeLastTwoWeeks
    - LastPlayTime
    - Language
    - Review
    - TimeCreated
    - VoteUp
    - VoteFunny
    - CommentCount
    - SteamPurchase
